HTML is the standard markup language for Web pages.
With HTML you can create your own Website.
HTML is easy to learn - You will enjoy it!

HTML 배우러 가기


HTML은 웹 페이지가 어떻게 표현되는지를 결정하는 마크업 언어로, HTML의 문법은 태그와 속성으로 구성되어 있다.
위의 영상을 통해 태그가 무엇인지 배워 보자.
HTML은 CSS, javascript와 함께 프론트엔드를 구현하기 위해 필요하다. 기본적인 웹 페이지는 HTML만으로 만들 수 있지만,
웹 페이지를 잘 꾸미기 위해서는 CSS가 필요하고 사용자와 상호작용하는 동적 웹을 만들기 위해서는 javascript가 필수적이다.
